SYARAT-SYARAT DAN KETENTUAN UMUM BERLANGGANAN

LAYANAN INTERNET CLass Home — PT. Jaringan Teknologi Sejahtera

Syarat-syarat dan Ketentuan Umum Berlangganan (“SKU”) ini adalah perjanjian antara PT. Jaringan Teknologi Sejahtera (“CLass Home”) dengan pelanggan CLass Home (“Pelanggan”). Perjanjian ini berlaku efektif sejak tanggal ditandatangani oleh CLass Home dan Pelanggan.

CLass Home dan Pelanggan selanjutnya disebut “Para Pihak”.

Pasal 1 – Definisi

  1. Perjanjian: Formulir Aplikasi Berlangganan (SKK + SKU) yang ditandatangani CLass Home dan Pelanggan.
  2. Layanan Telekomunikasi: layanan yang disediakan CLass Home dan/atau mitra (mis. akses internet).
  3. SKK: syarat & ketentuan khusus pada formulir pendaftaran.
  4. SKU: syarat & ketentuan umum (dokumen ini).
  5. Rupiah / Rp: mata uang Republik Indonesia.
  6. Terminal Telekomunikasi: perangkat ujung jaringan milik CLass Home.

Pasal 2 – Permohonan Berlangganan

  1. Permohonan dipenuhi bila calon Pelanggan memenuhi syarat teknis & administratif CLass Home.
  2. Calon Pelanggan wajib mendaftar melalui saluran resmi dan melampirkan identitas sah.
  3. Calon Pelanggan harus badan hukum yang sah atau perorangan ≥18 tahun (atau mendapat persetujuan wali).
  4. Data pendaftaran harus benar; Pelanggan bertanggung jawab atas kebenaran informasi.
  5. Masa berlangganan minimal 6 (enam) bulan dan otomatis diperpanjang untuk periode yang sama kecuali diakhiri sesuai Pasal 4.
  6. CLass Home berhak menolak permohonan tanpa perlu menjelaskan alasan.

Pasal 3 – Pemasangan Baru

  1. Pemasangan baru dipenuhi jika calon Pelanggan memenuhi persyaratan CLass Home.
  2. Pelanggan wajib hadir pada jadwal pemasangan; izin lokasi (apartment/gedung) dan biaya perizinan menjadi tanggung jawab Pelanggan.
  3. CLass Home menjelaskan proses pemasangan; Pelanggan wajib memberi izin dan membebaskan CLass Home dari tuntutan terkait pelaksanaan instalasi.
  4. Peralatan penyambungan (ONT, router, perangkat WiFi, dsb.) adalah milik CLass Home kecuali yang dimiliki Pelanggan.
  5. Peralatan yang disediakan Pelanggan menjadi tanggung jawab Pelanggan dan dapat mempengaruhi kualitas layanan.
  6. Pelanggan wajib menjaga peralatan CLass Home dan menandatangani berita acara penyerahan perangkat.
  7. Pembayaran tidak dilakukan secara tunai ke teknisi; gunakan saluran pembayaran resmi CLass Home.
  8. Material tambahan di luar instalasi standar akan dikenakan biaya tambahan sesuai ketentuan CLass Home.

Pasal 4 – Pemutusan Layanan

  1. Penunggakan dapat berakibat isolir atau pemutusan layanan dan penarikan perangkat yang dipinjamkan.
  2. Untuk tidak memperpanjang, Pelanggan wajib memberitahu CLass Home selambat-lambatnya 14 hari kalender sebelum tanggal cetak tagihan.
  3. Pelanggaraan Perjanjian memberi hak CLass Home memutus layanan.
  4. Pengembalian perangkat wajib dalam kondisi baik; biaya penggantian dikenakan bila rusak/hilang.
  5. Pindah lokasi wajib diberitahukan 14 hari kerja sebelumnya.
  6. Pemutusan tidak menghapus kewajiban pelunasan tunggakan.

Pasal 5 – Pembayaran Tagihan (Billing)

  1. Pelanggan membayar sesuai tanggal jatuh tempo yang diinformasikan CLass Home.
  2. Sistem pembayaran: pembayaran di muka (prepaid).
  3. Ketentuan denda keterlambatan dapat ditetapkan & diubah oleh CLass Home; CLass Home dapat mempertimbangkan kembali besaran atau penerapan denda.
  4. Semua pajak dipatuhi sesuai peraturan perpajakan RI.
  5. Pelanggan dilarang memberi gratifikasi kepada karyawan CLass Home.

Pasal 6 – Penagihan (Collection)

  1. CLass Home melakukan upaya penagihan sesuai hukum termasuk telepon, surat peringatan, atau kunjungan.
  2. Tim penagihan/pihak resmi diberi izin untuk menagih dengan mematuhi hukum.
  3. Pelanggan wajib bekerja sama dengan tim penagihan.

Pasal 7 – Layanan Pelanggan

  1. Pelanggan yang memenuhi kewajiban berhak mendapat layanan 24/7, kecuali Force Majeure.
  2. CLass Home dapat memberikan kompensasi bila gangguan >24 jam sesuai kebijakan.
  3. Layanan perbaikan ke lokasi dapat dikenakan biaya sesuai ketentuan.
  4. Perubahan paket tunduk pada ketentuan; downgrade tidak dikenakan biaya administrasi (kecuali memerlukan kunjungan/material).

Pasal 8 – Batasan Penggunaan

Pelanggan dilarang melakukan tindakan yang merugikan, antara lain (ringkasan):

  • Modifikasi jaringan/perangkat tanpa izin;
  • Menjual ulang layanan tanpa izin;
  • Spamming, hacking, pemalsuan identitas;
  • Penyebaran hoaks, SARA, atau tindakan ilegal;
  • Penggunaan layanan di luar wilayah bila dibatasi.

Pasal 9 – Pembatasan Tanggung Jawab

  1. Tanggung jawab CLass Home maksimum 10% dari biaya 1 bulan layanan dalam periode 12 bulan.
  2. CLass Home tidak bertanggung jawab atas kerugian yang bukan akibat kesalahan CLass Home (mis. Force Majeure, kesalahan pelanggan, interkoneksi).
  3. CLass Home tidak bertanggung jawab atas kehilangan data atau pendapatan pelanggan akibat penggunaan layanan.

Pasal 10 – Force Majeure

  1. Force Majeure mencakup peristiwa di luar kendali: putus fiber, bencana, kerusuhan, wabah, kebijakan pemerintah, dll.
  2. Pada Force Majeure, kewajiban CLass Home dapat ditangguhkan sampai keadaan normal kembali.

Pasal 11 – Sanksi

  1. CLass Home berhak mengenakan sanksi atas pelanggaran Perjanjian.
  2. Pelanggaran berat dapat dikenakan denda dan/atau pemutusan layanan.
  3. Denda pemutusan dini: sekurang-kurangnya Rp500.000,- apabila Pelanggan berhenti sebelum 6 bulan; besaran akhir dapat disesuaikan oleh CLass Home mempertimbangkan paket dan sisa masa kontrak.

Pasal 12 – Biaya-biaya

  1. Biaya downgrade dihapus; namun biaya kunjungan / material tetap berlaku bila diperlukan.
  2. Biaya kunjungan teknisi dikenakan untuk moving, dismantle, perbaikan akibat kelalaian, konfigurasi khusus, perapihan kabel, dll. Tarif diinformasikan CLass Home.
  3. Biaya penggantian perangkat (hilang/rusak karena kelalaian) dikenakan sesuai harga perangkat.
  4. Biaya tambahan material di luar instalasi standar akan dibebankan sesuai ketentuan CLass Home.

Pasal 13 – Hukum & Penyelesaian Perselisihan

  1. Perjanjian diatur oleh hukum Republik Indonesia.
  2. Perselisihan diselesaikan musyawarah; jika gagal, CLass Home berhak menempuh jalur hukum di pengadilan yang dipilih CLass Home.

Pasal 14 – Lain-lain

  1. Biaya tambahan belum termasuk PPN; dapat berubah sewaktu-waktu.
  2. CLass Home tidak bertanggung jawab atas materi pihak ketiga.
  3. CLass Home berhak melakukan promosi kepada Pelanggan.
  4. CLass Home dapat merevisi SKU; versi terbaru mengikat.
  5. Perangkat harus dikembalikan saat berhenti; jika tidak, biaya penggantian berlaku.
  6. Pelanggan wajib memberitahukan perubahan data dalam 14 hari kerja.
